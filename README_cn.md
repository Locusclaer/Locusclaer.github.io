# 翟一航个人主页

[English](README.md)

这是翟一航个人学术主页的源码仓库。

## 技术栈

- Next.js 静态导出
- TypeScript
- Tailwind CSS
- Markdown / TOML 内容文件

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

静态文件会输出到 `out/` 目录。

## 内容目录

- 站点配置：`content/config.toml`
- 中文内容：`content_zh/`
- 英文内容：`content/`
- 静态资源：`public/`
