# Zhai Yihang Homepage

[中文](README_cn.md)

This repository contains the source code for Zhai Yihang's personal academic homepage.

## Tech Stack

- Next.js static export
- TypeScript
- Tailwind CSS
- Markdown / TOML content files

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The static site is exported to `out/`.

## Content

- Main site configuration: `content/config.toml`
- Chinese content: `content_zh/`
- English content: `content/`
- Public assets: `public/`
